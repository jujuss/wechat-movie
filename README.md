# Wechat Movie
Movie client based on wechat

# Deploy

1. Install dependencies

 `$ cd wechat-movie && make depend`

2. Setup schedule

 `$ make cron`

3. Setup PYTHONPATH

 `$ export PYATHONPATH=$PYTHONPATH:$(pwd)`

# Commands

- Run service:

 `$ bin/start.sh`

- Stop service:

 `$ bin/stop.sh`

- Restart service:

 `$ bin/restart.sh`
